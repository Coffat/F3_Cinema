# Product Images

Folder này chứa ảnh sản phẩm cho F3 Cinema.

## Cấu trúc
- Đặt ảnh sản phẩm tại đây với format: `product-{id}.jpg` hoặc `product-{id}.png`
- Ví dụ: `product-1.jpg`, `product-2.png`

## Kích thước khuyến nghị
- Kích thước: 400x400px hoặc 800x800px
- Định dạng: JPG, PNG
- Tỷ lệ: 1:1 (vuông)

## Cách sử dụng
1. Đặt ảnh vào folder này
2. Cập nhật database với đường dẫn: `images/products/product-1.jpg`
3. Hoặc sử dụng URL từ internet
